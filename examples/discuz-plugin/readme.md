### Usage ###

1. git clone https://github.com/yoyow-org/yoyow-node-sdk.git
2. cd yoyow-node-sdk/examples/yoyow-authorize
3. npm install
4. edit conf/config.js
5. npm start
6. install yoyowauth.zip in discuz.
7. setting "API Server" and "Platform Id"
